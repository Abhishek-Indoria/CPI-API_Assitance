# CPI-API_Assitance

This calculator is derived from the internal CPI Reference Document.

It is used to:
- Calculate CPI / API increases
- Validate ISP values
- Handle Initial, Renewal, Co-term Upsell, and Non-co-term Upsell scenarios

## Files
- index.html – Calculator UI
- chatbot.js – Calculation logic
- style.css – Styling
- guide.html – CPI / API Reference Guide

## Usage
1. Open index.html
2. Enter contract details
3. Click Calculate
4. Validate ISP if required

In case of conflict, the CPI Reference Document takes precedence.
